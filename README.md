# Dynamic Form Engine 

A schema-driven dynamic form engine built using Vanilla JavaScript.

This project demonstrates advanced frontend engineering concepts such as:

- Dynamic form rendering
- Validation engine
- Conditional field rendering
- Multi-condition logic (AND / OR)
- State management
- API integration
- Local storage persistence
- Dynamic field cleanup
- Responsive UI architecture

---

#  Features

##  Dynamic Rendering
Generate form fields using JSON schema.

##  Validation Engine
Supports:
- Required validation
- Min length
- Email validation
- Custom validators

##  Conditional Logic
Supports:
- Single condition rendering
- Multi-condition rendering
- AND / OR logic

##  State Management
Custom lightweight state store implementation.

##  Local Storage Persistence
Form state survives page refresh.

##  API Integration
Async form submission using Fetch API.

##  Responsive UI
Mobile-friendly responsive design.
